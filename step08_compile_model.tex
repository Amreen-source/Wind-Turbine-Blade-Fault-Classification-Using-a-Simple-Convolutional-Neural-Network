\subsection*{Explanation}

\begin{itemize}
    \item \texttt{optimizer='adam'} --- The Adam optimiser adapts the learning rate automatically. It is a reliable default choice for most deep learning tasks.
    \item \texttt{loss='binary\_crossentropy'} --- The correct loss function for binary (2-class) classification problems.
    \item \texttt{metrics=['accuracy']} --- Tracks classification accuracy during training and validation.
    \item \texttt{model.summary()} --- Prints the full architecture showing each layer, its output shape, and parameter count.
\end{itemize}
