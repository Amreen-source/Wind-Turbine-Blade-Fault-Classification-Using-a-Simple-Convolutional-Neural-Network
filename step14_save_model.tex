\section*{Step 14 --- Save the Model}

\begin{lstlisting}[style=pythonstyle]
model.save("simple_wind_turbine_cnn.h5")
print("Model saved successfully.")
\end{lstlisting}

\subsection*{Output}

\begin{lstlisting}[style=outputstyle]
Model saved successfully.
\end{lstlisting}

\subsection*{Explanation}

This final step saves the entire trained model --- including its architecture, weights, and training configuration --- to a file called \texttt{simple\_wind\_turbine\_cnn.h5}.

The \texttt{.h5} format (HDF5) is a standard format for storing Keras models.

\subsection*{Why Saving the Model Is Important}

\begin{itemize}
    \item You do not need to retrain the model every time you want to use it.
    \item The saved model can be loaded later to make predictions on new images.
    \item It can be shared with others or deployed in an application.
\end{itemize}

\subsection*{How to Load the Model Later}

\begin{lstlisting}[style=pythonstyle]
from tensorflow.keras.models import load_model

loaded_model = load_model("simple_wind_turbine_cnn.h5")
\end{lstlisting}
