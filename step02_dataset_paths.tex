\section*{Step 2 --- Set Dataset Paths}

\begin{lstlisting}[style=pythonstyle]
train_dir = "/kaggle/input/datasets/mohammadshekaramiz/small-wind-turbine-blade-dataset-cai-swtb/CAI-SWTB-Dataset/Train"
val_dir   = "/kaggle/input/datasets/mohammadshekaramiz/small-wind-turbine-blade-dataset-cai-swtb/CAI-SWTB-Dataset/Validation"
test_dir  = "/kaggle/input/datasets/mohammadshekaramiz/small-wind-turbine-blade-dataset-cai-swtb/CAI-SWTB-Dataset/Test"
\end{lstlisting}

\subsection*{Explanation}

These lines tell Python where the images are stored:

\begin{itemize}
    \item \texttt{train\_dir} $\rightarrow$ training images
    \item \texttt{val\_dir} $\rightarrow$ validation images
    \item \texttt{test\_dir} $\rightarrow$ test images
\end{itemize}
