\section*{Step 13 --- Show Example Predictions}

\begin{lstlisting}[style=pythonstyle]
images, labels = next(test_data)
preds = model.predict(images)
preds = (preds > 0.5).astype(int).reshape(-1)

index_to_class = {v: k for k, v in test_data.class_indices.items()}

plt.figure(figsize=(12, 8))
for i in range(9):
    plt.subplot(3, 3, i + 1)
    plt.imshow(images[i])
    true_label = index_to_class[int(labels[i])]
    pred_label = index_to_class[int(preds[i])]
    plt.title(f"True: {true_label}\nPred: {pred_label}")
    plt.axis("off")
plt.tight_layout()
plt.show()
\end{lstlisting}

\subsection*{Explanation}

This block displays 9 test images in a $3 \times 3$ grid. Each image shows:
\begin{itemize}
    \item \textbf{True:} --- the actual label from the dataset.
    \item \textbf{Pred:} --- the label predicted by the model.
\end{itemize}

Images where \texttt{True} and \texttt{Pred} match are \textit{correct predictions}. Images where they differ are \textit{misclassifications}.

This visual check lets you see at a glance where the model performs well and where it struggles.
